# 演示录制建议

没有开发板时，可使用 `demo/index.html` 完成初赛演示录屏。桌面演示按官方 Bandx 手环示例的竖屏手环形态靠拢，目标比例参考 `194*368`。

## 操作脚本

1. 打开 `demo/index.html`，停留在表盘页面，展示当前时间和日期。
2. 向左滑动或点击右侧按钮，切换到心率页面，等待 2 秒展示模拟刷新。
3. 再次向左切换到计步页面，展示步数、目标进度和状态。
4. 向右滑动返回心率，再返回表盘，证明页面切换正常。
